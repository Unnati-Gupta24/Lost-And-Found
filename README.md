## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Unnati-Gupta24/Lost-And-Found.git
```

2. Install dependencies:

```bash
npm install
```

3. Build and run:

```bash
npm run build
```

```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
